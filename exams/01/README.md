# Exam 1

> Weight: 8%\
> Due: October 31

![Flag](https://upload.wikimedia.org/wikipedia/commons/9/99/Flag_of_Marinid_and_Saadi_Dynasty_%281258-1420%29_%281554-1659%29.svg)

Your task for this exam is to implement a program that displays the
above flag. We must be able to change the overall size of the flag by
changing the value of a single binding.

> [!NOTE]
> While you may use any function from Pyret's image library, the ones
> covered in class (`rectangle`, `overlay`, `rotate`, etc.) should be
> sufficient.

## Submission

To submit your assignment, you must first [click here][Classroom] and
follow the instructions. Once you have created the GitHub repository for
this assignment, you can use GitHub's web interface to upload your
program. Make sure to name the file "exam.arr".

[Classroom]: https://classroom.github.com/a/fkBy1WxX

## Assessment criteria

- Program design [5]
  - requirements are met
  - common code is unified, not duplicated
  - appropriate algorithms are used, and coded cleanly

- Readability [5]
  - constants are used instead of hard-coded values
  - complex or meaningful expressions are named
  - naming is consistent and descriptive
  - inline comments are used only to explain reasoning
  - lines are no longer than 80 characters
  - arguments are aligned vertically
  - use of whitespace is consistent
